---
description: Continue executing an existing Superpowers implementation plan
---

<SKILL_REQUIREMENT>

Using the skill tool, load the "superpowers:executing-plans" skills.

Check if there is an existing Superpowers implementation plan that has been executed but not completed.
Or user can specify a plan that has been executed but not completed.
If such a plan exists, continue executing it until completion.
If there are no such plans, inform the user that there are no ongoing Superpowers implementation plans to continue.
If there are multiple such plans, ask the user to choose which one to continue executing.

After finding such a plan, review the current status of the plan, identify the next steps to be taken, and execute those steps using the appropriate skills until the plan is completed.

</SKILL_REQUIREMENT>
